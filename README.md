# To-Do List

A simple CRUD to-do list app built with HTML, CSS, and JavaScript.

## Features
- Add new tasks
- Delete tasks
- Mark tasks as complete
- Filter tasks by All, Active, and Done

## How to Run
1. Clone the repository
2. Open `index.html` in your browser
